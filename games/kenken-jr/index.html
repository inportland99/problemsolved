How to Play
  1. Fill in each cell with a number from 1 to 4
  2. Every row and column contains each number exactly once
  3. The cells in a colored region combine to make the target value using given operation (+ or -)
    Note: Numbers in a colored region may occur more than once as long as they don't violate rule #2

KenKen Jr. uses only addition (+) and subtraction (-) operations!

Use the number tiles below the puzzle or keyboard to enter numbers into the grid. You may use pencil mode () to mark potential numbers in a cell
