Welcome to Shape Sum!

Your Goal:

Each shape represents a number (1-9). Use logic and the row/column sums to figure out which shape equals which number!

The Puzzle:

  • Each row's sum is shown on the right
  • Each column's sum is shown at the bottom
  • One shape or one sum is hidden - that's your mystery to solve!

How to Play:

  • Click the mystery shape to cycle through options
  • Use the Notepad below to track possible values for each shape
  • Lock in a shape's value when you're certain
  • When ready, click Check Puzzle

How to Use the Notepad

The notepad helps you track possible values for each shape:

  • Click numbers to cycle through three states:
    • Gray - Unsure
    • Green - Possible
    • Red - Not possible
  • Lock a value: When a shape has exactly one green value, click the lock icon to confirm it. This will:
    • Mark all other values for that shape as red
    • Display the locked value on all instances of that shape in the grid
    • Mark that value as red for all other shapes
  • Unlock: Click the lock again to reset that row
  • Reset Notepad: Clear all notes and locks at once
