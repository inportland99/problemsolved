What's a SET?

A SET is 3 cards where each attribute (number, shape, shading, color) is either all the same or all different.

Card Attributes

  • Number: 1, 2, or 3 shapes
  • Shape: Diamond, pill, or squiggle
  • Shading: Empty, striped, or solid
  • Color: Purple, blue, or magenta
