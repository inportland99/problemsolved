How to Play

Find all 6 SETs among the 12 cards!

Each unique card has four attributes (color, shape, number, shading). Each attribute has three possibilites.

Color: Purple, Blue, Magenta
Shape: Squiggle, Diamond, Pill
Number: 1, 2, 3
Shading: Solid, Striped, Open

A SET is 3 cards where each attribute is either all the same or all different.
