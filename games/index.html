Daily Math Games

Challenge yourself with a new puzzle every day!

KenKen

A daily 5×5 logic puzzle that combines sudoku-style rules with arithmetic operations. Fill the grid so each row and column contains 1-5, while making sure each cage's numbers produce the target using the given operation.

KenKen Jr.

A smaller 4×4 version of the classic KenKen puzzle, perfect for building logic skills with a simpler challenge.

Crack the Code

Use logic and deduction to crack the secret 3-digit code. Each guess gives you clues about which digits are correct and in the right position.

Shape Sum

A visual math puzzle where you assign numbers to shapes to make equations balance. Perfect for developing algebraic thinking.
